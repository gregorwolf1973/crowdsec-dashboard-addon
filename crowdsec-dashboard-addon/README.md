# CrowdSec Dashboard Add-on für Home Assistant

Web-UI zum Anzeigen und Verwalten von CrowdSec Bans und Alerts direkt in Home Assistant.

## Features
- 🔒 Alle aktiven Bans anzeigen (lokal & CAPI Community)
- ⚡ Alerts und erkannte Angriffe
- 🗑 Bans per Knopfdruck aufheben
- 🔍 Suche nach IP, Land oder Grund
- 🔄 Automatisches Refresh alle 30 Sekunden
- 🏠 Direkt in HA Sidebar integriert
